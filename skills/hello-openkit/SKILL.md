# Hello from OpenKit

This is the starter skill that ships with the OpenKit skills collection. It exists
so the repo is installable and testable from day one — replace or delete it once the
real skills land.

When this skill is invoked:

1. Briefly welcome the user to the OpenKit skills collection.
2. List the skills currently available in this collection (read the `skills/`
   directory if you have filesystem access, otherwise describe this one).
3. Point them at the README for install instructions and at CONTRIBUTING.md if they
   want to add their own skill.

Keep it short, friendly, and useful.
